# Dashboard Design

## Direcao visual

O dashboard adota uma linguagem inspirada em inteligencia territorial aplicada a saude animal:

- fundo claro com gradientes quentes e neutros
- alto contraste para leitura executiva
- uso de mapa e ranking como primeira camada de decisao
- graficos de drivers para explicar o indice, nao apenas exibir ranking

## Estrutura da interface

1. Header com contexto e framing do produto
2. Sidebar com pesos do indice e escopo por especie
3. Bloco superior com KPIs e mapa de oportunidade
4. Ranking executivo das UFs lideres
5. Scatter de drivers do indice
6. Serie trimestral por UF
7. Tabela analitica e manifesto de fontes

## Principios de UX

- cada visual deve responder a uma pergunta executiva distinta
- o indice composto nunca aparece sem os drivers
- os filtros devem alterar o indice de forma transparente
- os limites metodologicos precisam estar visiveis na narrativa do projeto
